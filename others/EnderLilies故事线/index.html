# Ender Lilies 故事线

这个故事要从这片大陆说起

不知道这片大陆最初是否有人居住，总之一个民族用咒术统治了世界并定居在这里.

这个民族会施展多种咒术，其中一种由咒术师施展的咒术能够产生一种名为 "污秽" 的物质，

这种物质一旦沾染上人体，便会侵蚀灵魂，使人体长出肉瘤，肉瘤使得人体具有惊人的再生能力，使肉体堪称不死之躯，但随着时间的流逝肉体会逐渐摆脱灵魂的控制，人也会因为污秽的怨念和杀意而发狂，继而丧失意识，无差别地对外界进行攻击，沦落为所谓的 "秽鬼"。

而另一种咒术就是巫女的净化

所谓净化，就是将秽鬼身上的污秽转移到活人身上从而解救秽鬼，如果一个人沾染污秽不久，肉体还没有长出很多肉瘤，那么被净化后还可以活命；如果沾染污秽很久，全身的肉都被肉瘤侵蚀，那么被净化后便会仅剩下一具白骨.

而巫女，便是古老民族中生来便对污秽有着一定抗性的女子，这种抗性保护着她们的灵魂使得在将污秽转移到自己身上后不会被污秽侵蚀.

当然巫女的承受力是有限的，随着巫女体内污秽越积越多，巫女会逐渐感到痛苦，最终因承受不住痛苦而遭到污秽侵蚀而沦为更可怕的 "秽鬼之王". 古老民族打造了一种守护宝具，藉由这种宝具，巫女在净化时的痛苦得以减轻。而这种宝具会随着净化的进行而逐渐减弱乃至破碎，需要用咒术重新修复宝具.

古老民族对抗污秽的第二种办法便是咒术契约印。这种仅由高级巫女才能发动的咒术通过令古老民族的战士向巫女献上灵魂契约从而舍弃肉体而灵魂永生不灭，契约印会在巫女的额头显现并随着巫女的家族代代相传，只要巫女不死或者契约不解除，古老民族的战士的灵魂便会不死。通过不死的战士，古老民族得以对抗秽鬼.

有一天，一群外族人大举进攻这片土地

虽然外族在人数上是优势，但在古老民族咒术契约印的加持下，战斗异常惨烈。在异端魔法师的帮助下，最终古老民族被打败，

这片土地被划分为六个区域，外族人之一被分到了大陆最东方的边陲之地，于是他就在这里建立了国家，是为第一代国王。然而，边陲之地的古老民族遗民始终没有屈服，幸存者藏身于边陲之地的地下，酝酿着污秽，伺机夺回故地.

渐渐地，生活在边陲之地的居民们开始发现王国里开始出现巨大的怪物，也就是秽鬼，秽鬼无论被打倒多少次都会重新站起来.

而第一代国王在从古老民族被屠尽的现场里救下了一个孩子，这个孩子便是古老民族的 "黎明的巫女宁菲莉雅". 因为宁菲莉雅身穿白衣，边陲之国的人们便称她为白巫女

白巫女宁菲莉雅净化了秽鬼从而解救了大家。正导教等许许多多教派迅速地从边陲之国消失，人们拥护在白巫女旗下，组成了白教.

白教主要活动在白教区和位于地下墓穴的地下修炼场。在白教中，白巫女拥有众多守护者，想要成为守护者就要在地下修炼场不断地修行.

# 第二代白巫女

后来黎明之白巫女宁菲莉雅在悬崖村生下了女儿并给予 "风" 的封号，也就是风之白巫女.

风之白巫女继承了黎明之白巫女的白巫女之位，肩负起净化秽鬼之责。都城的战士们与白巫女并肩作战最终击退秽鬼。于是风之白巫女在边陲之地的尽头建立了双生堡垒.

都城第一骑士格罗特在双生堡垒立下一块石碑，追悼与风之白巫女并肩作战的同伴.

# 第三代白巫女

后来风之白巫女的身体越来越虚弱，在边陲之泉生下女儿芙莉蒂雅并封号 "泉" 后便将泉之白巫女送去了故乡的悬崖村.

而后风之白巫女便由于承受不住污秽的侵蚀而被称为 "秽鬼研究第一人" 的异端魔术师法腾转移至地下的禁区进行名为 "治疗" 的研究，而法腾的助手兼爱人的迷瑞尔早在风之白巫女来到国家魔术师团之前就不幸沾染上了污秽

随着研究的深入，法腾发现取自风之白巫女身上的污秽肉瘤形状和性质都与取自秽鬼身上的不同，它的侵蚀速度更慢且对意识的影响被抑制了。于是法腾利用这一点制成了不死灵药.

随着泉之白巫女的长大，污秽大军在由深红色的污秽之王的率领下再次袭来。作为历代最强的白巫女，泉之白巫女率领众多骑士在双生堡垒迎战，战斗异常惨烈。骑士士兵罪犯甚至魔法师都被派往双生堡垒.

为了对抗污秽大军，双生堡垒的骑士和士兵包括骑士长尤利乌斯和狂骑士狼以及战士格罗特在内的众人都服用了不死灵药.

最终泉之白巫女以巫女历代相传的守护宝具破碎和身体被麻痹为代价成功净化秽鬼之王.

国家魔术师团于是将泉之白巫女转到地下的禁区进行治疗。由于芙莉蒂雅不想要孩子，也就没有继任者.

于是国家魔法师团以泉之白巫女为母体复制了八个白巫女，妄想将污秽转移到这些幼小的白巫女身上从而减轻芙莉蒂雅的痛苦，但却遭到了芙莉蒂雅的反对.

为此，芙莉蒂雅分别托西丽德，格罗特，伊莱恩，西丽亚，狂骑士狼，海尼尔，和国王分别照顾一个小白巫女.

最终，由于芙莉蒂雅支撑不住，在地下的污秽全部爆发，化作死亡之雨淹没了整个王国
